🎖️ Contexte : Lutte contre le terrorisme

Suite à la recrudescence des actes de criminalité orchestrés par les terroristes dans la république de Tognon, le Ministre de la défense a décrété 100 jours pour traquer ces hors la loi dans certaines zones du pays.

Dans le but de bien mener cette opération, le Ministre a mis un important lot de matériels à la disposition de l'ensemble des militaires. Ces matériels sont entre autres des véhicules pick-up et drones à vision nocturne. Il a notifié à ces militaires que le nombre de terroristes à traquer au cours des 100 jours devrait suivre une progression donnée par la suite (Uₙ)ₙ∈ℕ telle que :

U₁ = 5
U_(n+1) = U_n + 2

Le nombre total de terroristes à traquer est N = (1/10)T avec T = U₁ + U₂ + ... + U₁₀₀.

Tobi, un officier de la hiérarchie militaire, se préoccupe de connaître le nombre de terroristes à traquer au terme des 100 jours, le nombre de grandes zones concernées par l'opération et le nombre de pick-up et drones offert par le Ministre.

🎯 Tâche : Tu vas aider Tobi en résolvant les deux problèmes suivants.
🧩 Problème n°1
1) Détermine le nombre de terroristes traqués le 3ème jour, le 11ème jour puis le 100ème jour de l'opération.
2) Justifie que la suite (Uₙ) est une suite arithmétique de raison r = 2.
3) Détermine le nombre de terroristes traqués au terme de l'opération.
📐 Problème n°2

Le nombre de zones concernées par l'opération est une solution de l'équation t² − 3t − 10 = 0. Aussi, le triple du nombre de pick-up augmenté du double du nombre de drones est égal à 6800, puis le nombre de pick-up augmenté du triple du nombre de drones est égal à 4600.

4-a) Résous dans N, l'équation t² − 3t − 10 = 0.
4-b) Déduis-en le nombre de zones du pays concernées par l'opération.
5) On désigne par a le nombre de pick-up et par b le nombre de drones. Donne le système d'équations que vérifient a et b.
6-a) Résous dans R×R, le système suivant : (S) : { 3x + 2y = 6800 ; x + 3y = 4600 }
6-b) Détermine le nombre de pick-up et le nombre de drones offert par le Ministre.
🔒 Corrigé-type

🔐 Le corrigé-type est verrouillé. Soumets d'abord tes réponses ou entre le code fourni par ton professeur pour le débloquer.

1) Uₙ = U₁ + (n−1)×2 = 2n + 3.
U₃ = 9 ; U₁₁ = 25 ; U₁₀₀ = 203.
2) U_(n+1) − Uₙ = 2 (constante) pour tout n → la suite est bien arithmétique de raison r = 2, de premier terme U₁ = 5.
3) T = Σ Uₙ (n=1 à 100) = 100/2 × (U₁+U₁₀₀) = 50 × (5+203) = 10 400.
N = T/10 = 1040 terroristes traqués au terme de l'opération.
4-a) Δ = 9+40 = 49, √Δ = 7 → t = (3+7)/2 = 5 ou t = (3−7)/2 = −2. Dans N, seule t = 5 est retenue.
4-b) Le nombre de zones concernées est 5.
5) Système : { 3a + 2b = 6800 ; a + 3b = 4600 }
6-a) De x+3y=4600 → x=4600−3y. En remplaçant : 3(4600−3y)+2y=6800 → 13800−7y=6800 → y=1000 → x=4600−3000=1600.
S = {(1600 ; 1000)}
6-b) Nombre de pick-up = 1600 ; nombre de drones = 1000.
