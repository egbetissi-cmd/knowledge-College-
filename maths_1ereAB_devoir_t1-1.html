Problème 1 — Aires et équation du second degré

Q1a — Aire du trapèze (Figure 1)

Le trapèze a pour petite base x, grande base x+1, et hauteur x+3 (côté perpendiculaire aux deux bases).

Aire(trapèze) = ½ × (x + (x+1)) × (x+3) = ½ × (2x+1)(x+3)

En développant : = ½(2x² + 6x + x + 3) = ½(2x² + 7x + 3) = x² + 3,5x + 1,5

Q1b — Aire du rectangle (Figure 2)

Aire(rectangle) = x × (2x+3) = 2x² + 3x

Q2a — Justification de l'équation 2x²−x−3=0

Le partage est équitable si les deux aires sont égales :

½(2x+1)(x+3) = 2x² + 3x

(2x+1)(x+3) = 4x² + 6x

2x² + 7x + 3 = 4x² + 6x

0 = 2x² − x − 3

Donc x est bien solution de 2x² − x − 3 = 0.

Q2b — Résolution de 2x²−x−3=0

a=2, b=−1, c=−3

Δ = (−1)² − 4(2)(−3) = 1 + 24 = 25 = 5²

x = (1±5)/4 → x₁ = 6/4 = 3/2 ou x₂ = −4/4 = −1

Q2c — Valeur retenue

D'après la Figure 2, la contrainte est x > 0. On rejette donc x=−1.

La valeur retenue est x = 3/2 = 1,5

Problème 2 — Prix des parcelles

Q3 — Résolution de (E) : 2n⁴−n²−3=0

En posant t = n² (avec t≥0), l'équation (E) devient :

2t² − t − 3 = 0

C'est la même équation qu'au Problème 1 ! Δ=25, t = (1±5)/4 → t=3/2 ou t=−1.

Comme t=n²≥0, on rejette t=−1. Donc n² = 3/2

n = ±√(3/2) = ±(√6)/2

La solution positive est n = √6/2 ≈ 1,22.

Le prix de la parcelle de Zinsou est donc √6/2 million de francs (≈ 1,22 million de francs).

Q4a — Calcul de h(1)

h(1) = 2(1)³ − 3(1)² − 2(1) + 3 = 2 − 3 − 2 + 3 = 0

Q4b — Détermination de a, b, c

Puisque h(1)=0, (x−1) divise h(x). On développe (x−1)(ax²+bx+c) :

= ax³ + (b−a)x² + (c−b)x − c

Par identification avec h(x) = 2x³−3x²−2x+3 :

  • a = 2
  • b−a = −3 → b = −1
  • c−b = −2 → c = −3
  • −c = 3 → c = −3 ✓ (cohérent)

a=2, b=−1, c=−3, donc h(x) = (x−1)(2x²−x−3)

Q4c — Résolution de h(x)=0 et prix de Sagbo

h(x) = (x−1)(2x²−x−3) = 0

⟺ x−1=0 ou 2x²−x−3=0

x−1=0 → x=1

2x²−x−3=0 → x=3/2 ou x=−1 (déjà résolu au Problème 1)

Solutions de h(x)=0 : x ∈ {1 ; 3/2 ; −1}

La solution entière est x = 1.

Le prix de la parcelle de Sagbo est donc 1 million de francs.
