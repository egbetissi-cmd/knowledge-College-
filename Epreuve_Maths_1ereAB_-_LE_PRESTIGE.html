Contexte

Construction de l'immeuble « LE PRESTIGE ».
Le plan de la construction de l'immeuble « LE PRESTIGE » qui doit abriter un supermarché a été confié à l'architecte SEMEVO.

Pour établir le plan des divers compartiments de l'immeuble, il doit mettre sous forme canonique certaines expressions : P(x) = 6x² + x − 2 et q(x) = −2x² + 9x − 10. Par ailleurs, il doit aussi résoudre les équations et inéquations ci-dessous :

(E₁) : 6x² − 5x + 2 = 0     (I₁) : P(x) ≥ 0
(E₂) : x² − 3x + 9/4 = 0     (I₂) : q(x) < 0

Kossi retrouve enfin une expression C(x) = 4x² − 360x − 6300 qui permet de connaitre le coût journalier des travaux.

Kossi, un élève en classe de 1er AB, fils de SEMEVO, à son passage au cabinet de son père, découvre les plans. Attirés par certaines informations, qui s'y trouvent, il décide de vérifier ses acquis en mathématiques.

Tâche : Tu es un ami de Kossi, il te sollicite pour l'aider à améliorer ses acquis sur les notions données plus haut en résolvant les problèmes ci-dessous.

Problème 2

(S₁) : { x + 5y = 12 ; 4x − 3y = 2 }

(S₃) : { 1/x + 5(y+2) = 12 ; 4/x − 3(y+2) = 2 }
