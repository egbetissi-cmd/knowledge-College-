Problème 1 — Placements bancaires

Q1 — Intérêts des deux placements

Intérêt du 1er placement :

I₁ = 45 000 × t/100 = 450t

Capital réinvesti (capital + intérêt du 1er placement) :

C₂ = 45 000 + 450t

Intérêt du 2ème placement (taux (t+2)%) :

I₂ = C₂ × (t+2)/100 = (45 000+450t)(t+2)/100 = 4,5(100+t)(t+2)

Q2 — Équation t²+102t−880=0

On sait que I₂ = 4 860 :

4,5(100+t)(t+2) = 4 860

(100+t)(t+2) = 1080

100t + 200 + t² + 2t = 1080

t² + 102t + 200 − 1080 = 0

Soit t² + 102t − 880 = 0 ✓

Q3 — Résolution de t²+102t−880 > 0

Résolution de l'équation associée : Δ = 102² + 4×880 = 10404+3520 = 13924 = 118²

t = (−102±118)/2 → t₁ = 8 ou t₂ = −110

Le coefficient a=1>0, donc le trinôme est positif à l'extérieur des racines :

t² + 102t − 880 > 0 ⟺ t < −110 ou t > 8

Comme t représente un taux d'intérêt, t > 8 est la solution retenue dans le contexte.

Problème 2 — La parcelle rectangulaire

Q4a — Justification S(x)=x²+12x+11

S(x) = L × l = (x+11)(x+1) = x² + x + 11x + 11 = x² + 12x + 11 ✓

Q4b — Résolution de S(x)<0

x²+12x+11=0 : Δ = 144−44 = 100 = 10²

x = (−12±10)/2 → x=−1 ou x=−11

Le trinôme (a=1>0) est négatif entre les racines :

S(x) < 0 ⟺ −11 < x < −1

Cet intervalle ne concerne pas le contexte physique (x doit être strictement positif), il s'agit ici d'une résolution mathématique pure de l'inéquation posée.

Q5a — Détermination de L et l

L et l sont solutions de l'équation : X² − (L+l)X + L×l = 0

X² − 90X + 2000 = 0

Δ = 8100 − 8000 = 100 = 10²

X = (90±10)/2 → X=50 ou X=40

Comme L > l (longueur supérieure à largeur) : L = 50 dam et l = 40 dam

Q5b — Valeur de x et périmètre

L = x+11 = 50 → x = 39

Vérification : l = x+1 = 39+1 = 40 ✓ (cohérent)

Pourtour (périmètre) de la parcelle :

P = 2(L+l) = 2×90 = 180 dam
