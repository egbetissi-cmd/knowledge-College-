Problème 2 — Équations, inéquation et système

Q5 — Inéquation (I) : solutions [-2;2[

Domaine : (x+2)(3−x) ≥ 0 → x ∈ [-2;3]

Cas x < −1/2 : 2(2x+1) < 0 ≤ 3√(...) → inégalité toujours vraie sur [-2;-1/2[.

Cas x ≥ −1/2 : les deux membres positifs, on élève au carré : 4(2x+1)² < 9(x+2)(3−x) → développement donne une inéquation du 2nd degré dont la résolution, combinée au domaine, donne [-1/2;2[.

Réunion : [-2;-1/2[ ∪ [-1/2;2[ = [-2;2[ ✓

Q6a — Système (S₁) par pivot de Gauss

-3x-3y+2z=0 · 2x+3y+3z=-1 · x+2y-z=-1

Par élimination successive (pivot sur x dans L3) : x = -1, y = 0, z = -3/2 (à vérifier par substitution dans les 3 équations).

Q6b — Trois équations

(e₁'): x²−x−2=0 → Δ=9 → x = 2 ou x = -1

(e₂'): √(x+5)−x+1=0 → √(x+5)=x−1, domaine x≥1, élever au carré : x²−3x−4=0 → x=4 ou x=−1, seul x=4 convient (domaine).

(e₃'): |x−1|−2=0 → x−1=±2 → x = 3 ou x = -1

Q6c — Solutions dans ℝ³ du système (S)

En posant X=x−2/x, Y=√(y+5)−y, Z=|z−1|−2, le système (S) devient (S₁), résolu en Q6a.

Chaque valeur (X, Y, Z) trouvée est ensuite reliée aux racines des équations (e₁'), (e₂'), (e₃') de Q6b pour remonter à x, y, z entiers naturels cohérents avec le contexte (code d'accès).

Q7 — Code d'accès

D'après le triplet (α,β,γ) entiers naturels solution : le code s'écrit αβγαβγ en juxtaposant deux fois le triplet trouvé.
