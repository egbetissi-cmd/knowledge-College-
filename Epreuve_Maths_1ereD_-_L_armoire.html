Contexte

La fabrication d'une armoire.
Coffi est un élève en classe de 1ère D. Son tuteur FAKO lui demande d'aller retirer auprès du menuisier, l'armoire qu'il a commandée pour y ranger ses habits. Il dessine le plan de l'armoire sur une feuille de papier qu'il remet à Coffi pour faciliter l'identification chez le menuisier.

Voici le plan de l'armoire :

* ABCDEFGH est un pavé droit.
* ABFEIJKL est un cube.
* M, N, O, P sont les milieux respectifs des segments [LI], [JK], [GC], [HD].

Arrivé chez le menuisier, Coffi identifie l'armoire mais émet du doute sur certains parallélismes et orthogonalités.

Tâche : Résous les problèmes suivants pour apaiser Coffi.
Problème 1
1°) Démontre que : (AB) ⊥ (EH), (CJ) ⊥ (BAE). 1 pt
2°) Démontre que :
a°) les plans (GON) et (ABE) sont perpendiculaires. 1 pt
b°) les plans (BAL) et (JEF) sont perpendiculaires. 1 pt
3°)
a°) Démontre que BELJ est un rectangle. 1 pt
b°) Déduis-en que : (FA) ⊥ (LJ). 1 pt
Problème 2

Pour vérifier la justesse du positionnement de certains points sur l'armoire, Coffi s'intéresse aux projections orthogonales sur la droite (BF) et sur le plan (EFA).

Soit p la projection orthogonale sur la droite (BF).

4-a°) Détermine avec justification : p(K), p(B), p(J), p(C). 1,5 pt
b°) T est le point de l'espace tel que T = p(N). Justifie que FT = TB. 1 pt
5°) Le point F est le projeté orthogonal du point G sur le plan (EFA). Le point Q est le projeté orthogonal du point F sur la droite (BE). Démontre que : (GQ) ⊥ (BE). 1,5 pt
Problème 3

Le repérage de certains points de l'armoire a intéressé Coffi. Pour ce faire il choisit le quadruplet

R = (E ; ¼ FB ; ⅓ LK ; ½ EL)
6°) Justifie que R est un repère de l'espace. 1 pt
7°) Détermine dans R, les coordonnées des points B, J et M. 1 pt
8°) Détermine dans la base β = (¼ FB ; ⅓ LK ; ½ EL) les coordonnées des vecteurs KB et NL. 1 pt
9°) R, S et Z sont des points de l'espace tels que : JR(-2 ; 0 ; 1), JS(1 ; -1 ; 0), JZ(-1 ; -3 ; 2), AZ(½ ; 3/2 ; -1)
a°) Démontre que les points Z, J, R et S sont coplanaires. 1 pt
b°) Démontre que le point A appartient à la droite (JZ). 1 pt
c°) Démontre que les points J, R et Z déterminent un et un seul plan. 1 pt
10°) W est le point de l'espace vérifiant la relation : WF + WK + WL + WE = 0
a°) Démontre que : FW = ½ FK + ½ FE 1 pt

c°) Détermine dans R les coordonnées du point W. 1 pt

Repère de travail : le cube ABFEIJKL a pour arête a. En munissant l'espace du repère (A ; AB, AI, AE), on a A(0,0,0), B(a,0,0), F(a,0,a), E(0,0,a), I(0,a,0), J(a,a,0), K(a,a,a), L(0,a,a). Le pavé ABCDEFGH a pour profondeur d (direction -y) : C(a,-d,0), D(0,-d,0), G(a,-d,a), H(0,-d,a). On a alors M(0,a,a/2), N(a,a,a/2), O(a,-d,a/2), P(0,-d,a/2).
1°) AB(a,0,0) et EH(0,-d,0) : AB·EH=0 → (AB)⊥(EH). Le plan (BAE) est le plan {y=0} de normale (0,1,0) ; CJ=(0,a+d,0) est colinéaire à cette normale → (CJ)⊥(BAE).
2-a°) G,O,N ont tous pour abscisse x=a → le plan (GON) est le plan {x=a}, de normale (1,0,0). Le plan (ABE) est {y=0}, de normale (0,1,0). (1,0,0)·(0,1,0)=0 → les normales sont orthogonales, donc (GON)⊥(ABE).
2-b°) Une normale au plan (BAL) est AB∧AL = (0,-a²,a²), colinéaire à (0,-1,1). Une normale au plan (JEF) est EJ∧EF = (0,-a²,-a²), colinéaire à (0,-1,-1). (0,-1,1)·(0,-1,-1)=1-1=0 → (BAL)⊥(JEF).
3-a°) BE=(-a,0,a) et LJ=(a,0,-a)=-BE : côtés opposés égaux → BELJ est un parallélogramme. De plus BE·EL=(-a,0,a)·(0,a,0)=0 → un angle droit. Donc BELJ est un rectangle.
3-b°) D'après 3-a°, LJ est parallèle à BE (côtés opposés du rectangle). Or AF et BE sont les diagonales du carré ABFE, donc (AF)⊥(BE). Comme (LJ)∥(BE), on déduit (FA)⊥(LJ).
4-a°) La droite (BF) est la droite {x=a, y=0}. La projection orthogonale garde la cote z et ramène (x,y) à (a,0) : p(K)=F (K et F ont la même cote a), p(B)=B (B est déjà sur la droite), p(J)=B et p(C)=B (J et C ont pour cote 0, comme B).
4-b°) N, milieu de [JK], a pour cote a/2 (moyenne de 0 et a) ; donc T=p(N) a pour coordonnées (a,0,a/2), qui est le milieu de [BF]. Comme T est le milieu de [BF], FT=TB=a/2.
5°) Le plan (EFA) est le plan {y=0} ; le projeté de G(a,-d,a) sur ce plan est bien (a,0,a)=F, ce qui confirme la donnée. Le projeté orthogonal Q de F sur (BE) est le milieu de [BE] (le paramètre trouvé vaut 1/2), soit Q(a/2,0,a/2). On calcule GQ=(-a/2,d,-a/2) et BE=(-a,0,a) : GQ·BE = a²/2+0-a²/2 = 0. Donc (GQ)⊥(BE).
6°) Les vecteurs ¼FB=(0,0,-a/4), ⅓LK=(a/3,0,0), ½EL=(0,a/2,0) sont deux à deux non colinéaires (chacun porté par un axe différent) : ils forment une base de l'espace. Avec le point E comme origine, R=(E ; ¼FB ; ⅓LK ; ½EL) est bien un repère de l'espace.
7°) En résolvant EB, EJ, EM dans la base (¼FB, ⅓LK, ½EL), on trouve : B(4 ; 3 ; 0), J(4 ; 3 ; 2), M(2 ; 0 ; 2).
8°) De même pour les vecteurs KB=(0,-a,-a) et NL=(-a,0,a/2) : KB(4 ; 0 ; -2) et NL(-2 ; -3 ; 0) dans la base β.
9-a°) On cherche λ,μ tels que JZ=λJR+μJS : -1=-2λ+μ ; -3=-μ → μ=3 ; 2=λ → λ=2 ; on vérifie -2(2)+3=-1 ✔. Donc JZ=2JR+3JS : JZ est combinaison linéaire de JR et JS, donc Z, J, R, S sont coplanaires.
9-b°) AZ=(½ ; 3/2 ; -1) et JZ=(-1 ; -3 ; 2). On remarque AZ = -½·JZ : les vecteurs AZ et JZ sont colinéaires et ont le point Z en commun, donc A, Z et J sont alignés : A appartient à la droite (JZ).
9-c°) JR=(-2;0;1) et JZ=(-1;-3;2) ne sont pas colinéaires (sinon on aurait -1=-2k et -3=0k pour un même k, ce qui est impossible). J, R et Z ne sont donc pas alignés : ils déterminent un seul et unique plan.
10-a°) WF+WK+WL+WE=0. En écrivant WK=WF+FK, WL=WF+FL, WE=WF+FE, on obtient 4WF+FK+FL+FE=0. Or FKLE est un carré (parallélogramme) donc FL=FK+FE. D'où 4WF=-2(FK+FE), soit WF=-½(FK+FE), donc FW = ½FK + ½FE.
10-b°) FW=½(FK+FE)=½FL : W est donc le milieu de la diagonale [FL] du carré FKLE, c'est-à-dire le centre de la face carrée supérieure de l'armoire (le "toit" du cube).
10-c°) W=(a/2,a/2,a) dans le repère (A;AB,AI,AE). En résolvant EW=(a/2,a/2,0) dans la base (¼FB,⅓LK,½EL), on trouve W(0 ; 3/2 ; 1) dans le repère R.
