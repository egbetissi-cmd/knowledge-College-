Guide d'intégration rapide
Étape 1 — Configurez votre profil et créez un code de classe (ex: TERM-D-SVT-2026)
Étape 2 — Partagez le code de classe à vos élèves avant le test
Étape 3 — Chaque élève génère son code personnel dans l'Espace Élève
Étape 4 — Suivez en temps réel depuis le Tableau de Bord
Étape 5 — Les parents vérifient via l'Espace Parent avec le code de leur enfant
